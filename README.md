## Installation and Setup

Clone the repository:

```bash
git clone https://github.com/Santhoshi003/portfolio-website.git
```

Move into the project folder:

```bash
cd portfolio-website
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open browser:

```text
http://localhost:5173
```
